# Event Log Extras

This add-on is intended for Tacview Enterprise users in conjunction with the Event Log add-on.

Several features are added to supplement the Event Log:

* Right click on an object to Kill Remove it or to make it Alive.

* Double click on an event in the event log and then right-click in the 3D view to edit the event (Timeout, Message or Bookmark).  

* Update the files outcome-list.lua and ammo-list.lua to meet your needs.

## Limitations
